我关注那些会随时间复利的小系统：本地优先的助手、可复现的研究笔记、数学建模流水线，以及让下一次实验更容易启动的开发者工具。

### 当前主线

- 面向检索、记忆、规划和代码审查的 AI Agent 工作流。
- 数学建模，以及受拓扑方法启发的数据分析。
- 基于 Next.js、Quartz 和自动化脚本的个人知识基础设施。
- 长期使用时足够安静、可读、可靠的界面。
